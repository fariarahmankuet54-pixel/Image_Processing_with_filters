# Image Processing with Multiple Filters

This repository includes MATLAB implementations of different image filtering techniques in both spatial and frequency domains. This project was done as a part of Digital Image Processing graduate level course.

## Included methods
- 7×7 average filter
- DFT filtering
- Ideal low-pass filter
- Gaussian low-pass filter

## Analysis
- Mean Squared Error (MSE)
- Improvement in Signal-to-Noise Ratio (ISNR)

- MATLAB scripts
